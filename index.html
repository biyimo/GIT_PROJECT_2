GIT AND GITHUB

What is git?

Git is an opensource version control system that allows developer to track versions of their codes overtime

Git tracks our codes by creating snapshot of the code and can be revisited whenever necessery

CONFIGURING GIT

git config --global user.name "OLAMILEKAN" is used to configure our name on git

git config --global user.email "akinbiyimoses@gmail.com" is used to configure our email on git

BASIC COMANDS

git init is used to initialize git in your local repository.

Git status is used to check the status of the activites of git in your local repository.

git "FILENAME " is used to track a single file in git

git "add ."is used to track multiplue files in git

git commit -m "the commit mesasge is used when we want to save a snapshot on our code. it is like a caption or the description of the changes made tou our code

git rm --cached "filename" is used to untrack a file in git

git remote add origin "REPO URL" is used to connect you local repo to github

git push - u origin master is used to push our code from local repo to remote repository




What is GITHUB?

GitHub is a web base git repository hosting service

Github is a tool that enables collaborations by hosting shared git repositories that team of developers can all contribute to

while github uses git, the functionality it provides is entierly different from git

GIT is a version control system, While Github is an inline platform for hosting and sharing code

WHY USE GITHUB

  1. github provides a greatway to store your code in a safe and remote location.
  2. Many large open source projects are hosted on GITHUB
  3. Github encourages colaboration betwwen a team of developers.
